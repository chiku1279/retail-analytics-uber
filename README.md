# 🚖 Uber Data Analysis Dashboard

## 📌 Project Overview

This project presents a comprehensive analysis of Uber ride data using Power BI. The dashboard is designed to uncover key insights related to ride demand, customer behavior, and operational performance.

The report is divided into **three interactive pages**, each focusing on a different aspect of the business.

---

## 📂 Dataset Description

The dataset includes key ride-level details such as:

* Trip Date & Time
* Pickup & Drop-off Locations
* Ride Distance
* Fare Amount
* Payment Type
* Trip Duration

---

## 📊 Report Pages

### 1️⃣ Overview Dashboard

This page provides a high-level summary of the Uber dataset.

**Key Metrics:**

* Total Trips
* Total Revenue
* Average Fare
* Average Trip Distance

**Visuals Included:**

* KPI Cards
* Time-series trends (Trips & Revenue)
* Distribution of ride categories

**Purpose:**
To quickly understand overall performance and identify general trends.

### 2️⃣ Time Analysis

This page focuses on when and where demand is highest.

**Key Insights:**

* Peak hours and busiest days

* Ride frequency patterns

**Visuals Included:**

* Heatmaps (Time vs Demand)
* Bar charts (Trips by Hour/Day)
* Location-based visualizations

**Purpose:**
To identify demand patterns and optimize resource allocation.

This page dives deeper into operational and financial performance.
